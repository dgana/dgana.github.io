Get to Know Adaptive Web Design (AWD) and Responsive Web Design (RWD)

1. Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

Adaptive adalah desain web statis yang dikirim oleh server pada saat client masuk ke dalam website dan langsung disesuaikan dengan media yang digunakan. Sedangkan Responsive yaitu desain web yang lebih bersifat dinamis karena selalu mengikuti besaran dari media yang digunakan. Keduanya merupakan teknik yang bagus untuk desain sebuah web, tergantung tujuan website tersebut target audiencenya siapa saja. Jika pengunjung diperkirakan akan akses dari berbagai media maka kemungkinan besar adaptive lebih baik dibandingkan responsive. Jika perkiraannya pengunjung adalah desktop user lebih dari 70% maka lebih baik menggunakan RWD. Statistik menunjukkan bahwa lebih banyak web developer menggunakan teknik RWD. Mungkin karena lebih mudah karena tidak perlu membuat desain pada setiap screen size. Balik lagi tergantung tujuan/goals website itu sendiri.

2. Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

Ada perubahan signifikan pada desain sebuah web jika dilihat pada tahun 2010 masih belum banyak yang menggunakan smartphone dan tablet sedangkan pada tahun 2016 ini hampir semua orang menggunakan smartphone! Namun jika screen size yang digunakan di masa depan masih sama antara kisaran 320px - 1600px maka perubahan tidak terlalu signifikan menurut saya.

3. Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

Pada teknologi website saya mengenal yang namanya role pada HTML. Contoh role yaitu pada penggunaan bootstrap (framework CSS) pada navigation biasanya ada <nav role="navigation"></nav> yang digunakan untuk para penyandang disabilitas dalam penggunaan komputer. Lebih tepatnya untuk penggunaan screen reader yaitu app software yang mengeluarkan output berupa suara. Tentunya saya setuju jika ada teknologi baru untuk para screen reader.

4. Ide atau konsep apa yang sesuai denganmu saat mempelajari desain web?

Konsep penggunaan Responsive Web Design menurut saya lebih cocok jika target market tidak menggunakan terlalu banyak device dan pengerjaan juga lebih cepat. Penggunaan Adaptive Web Design lebih cocok jika target marketnya sangat luas dan mempunyai konsep yang berbeda antara pengguna mobile dan desktop user begitu juga tablet dll.
